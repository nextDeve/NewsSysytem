### 1.项目来源

[千锋2022项目实战]([千锋2022版React全家桶教程_react零基础入门到项目实战完整版_哔哩哔哩_bilibili](https://www.bilibili.com/video/BV1dP4y1c7qd?p=152))

### 2.项目功能

<ol>
    <li>登录模块
        <ul>
            <li>登录验证拦截</li>   
        </ul>
    </li>
    <li>权限管理模块
        <ul>
            <li>权限列表</li>
            <li>角色列表</li>
        </ul>
    </li>
    <li>用户管理模块
        <ul>
            <li>用户列表</li>
        </ul>
    </li>
    <li>新闻管理模块
        <ul>
            <li>富文本撰写新闻</li>
            <li>草稿箱</li>
            <li>新闻分类</li>
        </ul>
    </li>
    <li>审核管理模块
        <ul>
            <li>审核新闻</li>
            <li>审核列表</li>
        </ul>
    </li>
    <li>发布管理模块
        <ul>
            <li>待发布</li>
            <li>已发布</li>
            <li>已下线</li>
        </ul>
    </li>
     <li>简易新闻数据分析</li>
     <li>简易新闻展示</li>
</ol>
